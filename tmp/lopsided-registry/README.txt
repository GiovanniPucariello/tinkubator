lopsided-registry
================================================================================
Part of LoPSideD: an implementation of the Linked Process Protocol

To start the LoPSideD registry in Unix, use the Bash launcher script:

    $ bash registry-nogui.sh config.properties

where config.properties is a Java properties file containing the configuration
of your LoPSiDeD environment.
Similarly, in Windows you may start a registry using the provided batch script:

    $ registry-nogui config.properties

For more information on Linked Process and the LoPSideD implementation, see:
    http://linkedprocess.org
Questions and comments are always welcome on the Linked Process mailing list:
    linkedprocess@googlegroups.com

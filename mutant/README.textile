Mutant supports the use of multiple "JSR 223":http://jcp.org/en/jsr/detail?id=223 ScriptEngines over the same variable pool. In this way, its possible to make use of each language's unique expressive qualities when manipulating variables.

The documentation for Mutant can be found at this "location":http://mutant.tinkerpop.com. Finally, please visit "TinkerPop":http://tinkerpop.com for other software products.

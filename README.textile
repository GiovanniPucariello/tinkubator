LoPSideD is an implementation of the "Linked Process":http://linkedprocess.org protocol. This implementation can be used to create a distributed computing cloud that can be supported and leveraged by any device with an Internet connection.

The documentation for LoPSideD can be found at this "location":http://lopsided.tinkerpop.com. Finally, please visit "TinkerPop":http://tinkerpop.com for other software products.
